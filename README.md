# Slow Sort Implementation

This is a C++ implementation of the Slow Sort algorithm, a recursive sorting algorithm that sorts an array in ascending order.

## Description

Slow Sort is an inefficient sorting algorithm with a time complexity of O(n^(log n / log 2)), making it one of the slowest sorting algorithms. It works by recursively sorting the left and right halves of the array, then ensuring the middle and last elements are in order.


## Usage

- Enter the number of elements when prompted.
- Enter the elements separated by spaces.
- The program will display the original array and the sorted array in ascending order.
